## Полезные команды для взаимодействия с командной строкой(в основном в UNix)
* getEnv()-получение переменной окружения по имени
* setEnv()-установка переменной окружения по имени
* ls - вывести информацию об объектах ФС
* "-" -файл
* d - каталог
* touch filename.c - создание файла(установить временные характеритики на текущее время)
* cp filename1.c filename2.c - копирование
* mv filename1.c filename2.c -переместить/переименовать
* rm filename1.c filename2.c - удаление файлов через пробел
* mkdir directory_name - создание каталога
* cd directory_name  - переход в каталог dir1
* cd - домашний каталог пользователя
* rm -rf directory_name - удалить рекурсивно каталог с подавлением интерактивности.
* mc - файловый менеджер
* Ctrl+R  - релоад каталога
* Ctrl+O - скрыть панель
* Esc+:q! - закрыть vim
* shutdown -h now - незамедлительное выключение 
* F4 - встроенный редактор
* F2 - сохранить
* F10 - выйти
* cc -c filename.c - только скомпилировать файл
* cc filename.o - линковка
* . - текущий каталог
* .. - родительский каталог
* cc -o project_name filename.c - сразу создать бинарник
* ./project_name - запуск программы
